### 简易安装方法：

打开终端，执行下面的命令就自动安装好了：

`wget -qO- https://raw.github.com/progpark/mymacvim/master/install.sh | sh -x`

- 重新打开vim即可看到效果

### 了解更多vim使用的小技巧：

### 多窗口操作
1. 使用`:sp + 文件名`可以水平分割窗口
- 使用`:vs + 文件名`可以垂直分割窗口
- 使用`Ctrl + w`可以快速在窗口间切换
